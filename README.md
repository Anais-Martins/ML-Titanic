# ML-Titanic
Initiation au Machine Learning - Prédiction de survie sur le Titanic
Projet d'initiation au Machine Learning réalisé sur le dataset Kaggle Titanic.

## Objectif
Prédire quels passagers ont survécu au naufrage du Titanic à partir de données personnelles (âge, sexe, classe...).

## Démarche
1. Exploration des données - Analyse des variables et visualisations
2. Nettoyage des données - Gestion des valeurs manquantes, encodage des variables
3. Modélisation - Entraînement d'un modèle Random Forest
4. Évaluation - Accuracy de 82.68% sur le set de validation

## Résultats
Accuracy (validation) : 82.68 % 
Score Kaggle (test réel) : 73.92 % 

## Technologies
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
